
# OrangeHRM Automation (Selenium + POM)

## Features Automated:
- Login to OrangeHRM
- Navigate to PIM module
- Add new employees
- Verify employees in the employee list
- Logout

## How to Run:
1. Install dependencies:
```
pip install -r requirements.txt
```

2. Run test:
```
python tests/test_orangehrm_workflow.py
```
